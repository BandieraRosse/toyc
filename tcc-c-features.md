# tcc  — C 语言功能支持清单

> 基于 `compiler/tcc` 源码分析，测试套件 `compiler-tests/` 验证。
> 更新日期：2026-07-11

## 图例

| 符号 | 含义 |
|------|------|
| ✅ | 完全支持，有测试覆盖 |
| ◐ | 部分支持或有限制 |
| ⬜ | 语法上接收但无语义效果 |
| ❌ | 未实现 |
| — | 不适用 |

## 9. C 标准版本覆盖小结

| 标准 | 覆盖情况 |
|------|----------|
| **C89/C90** | **绝大部分覆盖**。缺少：`auto` 关键字、`#line` 指令 |
| **C99** | **核心特性基本覆盖**。支持：`//` 注释、声明与语句交错、`for` 内声明、`long long`、`inline`（语法）、`restrict`（语法）、变参宏、`__VA_ARGS__`。缺少：VLA、复合字面量、指定初始化器、`_Bool`、`_Complex`、`__func__`、`#line`、`#if` 常量表达式 |
| **C11** | **几乎未覆盖**。`_Alignas`、`_Alignof`、`_Noreturn`、`_Static_assert`、`_Atomic`、`_Thread_local`、`_Generic` 均未实现 |
| **GNU 扩展** | **有限支持**：内联汇编完整实现；`__attribute__` 语法接收但无效果；`extern "C"` 跳过；case range 支持。`__DATE__`/`__TIME__` 未定义 |
